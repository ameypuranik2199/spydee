# Spydarr Webtech - Smooth Web Weave

This project features a smooth, cinematic intro for Spydarr Webtech.

## Features
- GSAP-powered smooth movement.
- A stylized, childish spider character.
- Radial web weaving logic built with Canvas.
- Dark, professional theme.

## Deployment
1. Upload the `index.html` file to your GitHub repository.
2. Go to **Settings > Pages**.
3. Select your main branch and save.
